Displaying Username

Learning Objectives

  • Display the username of the user who created the issue on each Issue component.

Why Include & Show the Username?

  • Accountability & Context: See who reported/owns an issue at a glance.
  • Data Integrity: Attribute issues consistently and maintain a clear audit trail.
  • Querying & Views: Filter by user; support user-specific feeds and analytics.

Add username to the Issue Schema 


const mongoose = require('mongoose');
const Schema = mongoose.Schema;

const issueSchema = new Schema({
  title:       { type: String, required: true },
  description: { type: String, required: true },
  imgUrl:      { type: String },
  userId:      { type: Schema.Types.ObjectId, ref: "User" },
  username:    { type: String, required: true } // display name of creator
});

module.exports = mongoose.model('Issue', issueSchema);

Ensure your auth token includes username so you can set it from req.auth on creation.

Assign Username on Create (Router)


const express = require('express');
const issueRouter = express.Router();
const Issue = require('../models/issue');

issueRouter.post('/', async (req, res, next) => {
  try {
    req.body.username = req.auth.username; // from verified JWT
    req.body.userId   = req.auth._id;
    const newIssue    = new Issue(req.body);
    const savedIssue  = await newIssue.save();
    return res.status(201).send(savedIssue);
  } catch (error) {
    res.status(500);
    return next(error);
  }
});

module.exports = issueRouter;

Display Username in the UI (React)


import React from 'react';

function Issue(props) {
  const { title, description, imgUrl, username } = props;

  return (
    <article className="card">
      <header className="card-header">
        <h4 className="muted">{username}</h4>
        <h3>{title}</h3>
      </header>

      <p>{description}</p>
      {imgUrl && <img src={imgUrl} alt={title} className="responsive-img" />}
    </article>
  );
}

export default Issue;
