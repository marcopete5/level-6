Overview

MongoDB’s update operators let you efficiently modify document fields and arrays—perfect for features like voting, tagging, counters, and more. Below we pair a practical voting example with a quick-reference of common operators.

Issue Schema (with vote arrays)


const mongoose = require("mongoose");
const Schema = mongoose.Schema;

const issueSchema = new Schema({
  title:       { type: String, required: true },
  description: { type: String, required: true },
  imgUrl:      { type: String },
  userId:      { type: Schema.Types.ObjectId, ref: "User" },
  username:    { type: String, required: true },
  upvotes:     [{ type: Schema.Types.ObjectId, ref: "User" }],
  downvotes:   [{ type: Schema.Types.ObjectId, ref: "User" }]
});

module.exports = mongoose.model("Issue", issueSchema);

Upvote Route


issueRouter.put('/upvotes/:issueId', async (req, res, next) => {
  try {
    const updatedIssue = await Issue.findByIdAndUpdate(
      req.params.issueId,
      {
        $addToSet: { upvotes: req.auth._id }, // add current user if not already present
        $pull:     { downvotes: req.auth._id } // ensure user is removed from downvotes
      },
      { new: true }
    );
    return res.status(201).send(updatedIssue);
  } catch (error) {
    res.status(500);
    return next(error);
  }
});

async function handleUpvote(issueId) {

Common MongoDB Update Operators

  • $set — set a field’s value. 
    { $set: { status: "completed" } }
  • $unset — remove a field. 
    { $unset: { age: "" } }
  • $inc — increment/decrement numeric fields. 
    { $inc: { likes: 1 } }
  • $addToSet — add to array if not already present. 
    { $addToSet: { tags: "mongodb" } }
  • $pull — remove matching entries from an array. 
    { $pull: { tags: "mongodb" } }
  • $push — append to an array (optionally with modifiers). 
    { $push: { comments: { text: "Nice post!" } } }
  • $pop — remove first or last array element. 
    { $pop: { items: 1 } }  // last
{ $pop: { items: -1 } } // first
